# Interface: QueryParams

parameters to define a query

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="columns"></a> `columns` | `string`\[] | array of column names to retrieve values for |
| <a id="limit"></a> `limit?` | `number` | number of rows to return |
| <a id="orderby"></a> `orderBy?` | `string` | column name to sort records by in ascending order |
| <a id="orderbydesc"></a> `orderByDesc?` | `string` | column name to sort records by in descending order |
| <a id="query"></a> `query` | `string` | encoded query string |
| <a id="table"></a> `table` | `string` | table name to query |
