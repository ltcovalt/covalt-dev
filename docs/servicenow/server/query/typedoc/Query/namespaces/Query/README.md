# Query

Server-side namespace containing methods to execute secure `GlideRecordSecure` queries,
returning a plain, sanitized JavaScript object containing only the requested columns and values.

## Functions

| Function | Description |
| ------ | ------ |
| [records](functions/records.md) | Executes a GlideRecord query and returns an array of objects containing only the requested columns. Can also be used to retrieve dotwalked values. |
