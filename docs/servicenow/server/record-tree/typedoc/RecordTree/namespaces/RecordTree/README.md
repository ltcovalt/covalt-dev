# RecordTree

namespace containing tree traversal functions

## Functions

| Function | Description |
| ------ | ------ |
| [getRootNode](functions/getRootNode.md) | Returns the root node of a given node |
| [traversePostOrder](functions/traversePostOrder.md) | Traverses an n-ary tree using a post-order depth first search |
| [traversePreOrder](functions/traversePreOrder.md) | Traverses an n-ary tree using a pre-order depth first search |
| [traverseToRoot](functions/traverseToRoot.md) | Visits each node in a tree structure, starting from the given node and moving up to the root node |
